## Arquitectura del Repositorio

La carpeta raíz `Project PLAN DO` se distribuye bajo el siguiente esquema modular:

*   `index.html` → Estructura y esqueleto base de la Landing Page (One-Page Scroll).
*   `css/` → Módulos de diseño visual independientes:
    *   `main.css` → Variables de color corporativo, tipografías globales y reseteos.
    *   `navbar.css` → Diseño del menú de navegación superior fijo.
    *   `hero-intro.css` → Estilos de la cabecera de impacto y la sección introductoria (Contexto local de BioPago, Pago Móvil y WhatsApp Business)[cite: 1].
    *   `cards.css` → Estilos en Grid/Flexbox para bloques de Misión/Visión y los 4 Casos de Estudio (Antipatrones de seguridad).
    *   `matrix.css` → Maquetación adaptada para la tabla de planificación de 6 fases requerida metodológicamente.
*   `js/` → Lógica de comportamiento e interactividad del sitio:
    *   `main.js` → Orquestador e inicializador general de los scripts.
    *   `navigation.js` → Control de scroll suave (*smooth scroll*) y efectos del menú.
    *   `dynamic-content.js` → Almacenamiento e inyección dinámica de los textos y casos de estudio para mantener el HTML limpio.
    *   `matrix-accordion.js` → Comportamiento del "acordeón" interactivo para desplegar objetivos y actividades de cada fase de forma animada.
*   `assets/docs/` → Carpeta destinada exclusivamente a almacenar el documento original
