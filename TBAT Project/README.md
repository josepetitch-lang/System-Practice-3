3) Conceptos Utilizados
-- TGS (Teoría General de Sistemas): Dentro de un sistema, el todo le gana al comportamiento individual de sus componentes
#3.1 Conceptos de TGS Utilizados:
-- Entropía: Es la tendencia natural de cualquier sistema hacia el desorden 
-- Neguentropía: Es el proceso mediante el cual un sistema logra contrarrestar la entropía
-- Homeostasis: Es la capacidad de un sistema para mantener un equilibrio interno estable
